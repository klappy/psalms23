# Psalm 23 - The Lord is My Shepherd
## Immersive Visual Experience

A stunning, interactive 3D visualization of Psalm 23 designed to bring the beloved psalm to life through modern web technology. This experience combines beautiful visuals, smooth animations, and responsive design to create an immersive journey through each verse.

## Features

### 🎨 Visual Design
- **3D Environments**: Each verse features a unique 3D scene representing its imagery
- **Dynamic Lighting**: Lighting changes to match the mood of each verse
- **Particle Effects**: Heavenly particles, water ripples, and golden cup overflow
- **Scrolling Background**: Roku-style scrolling biblical imagery (crosses, doves, stars, clouds)
- **Sky Gradients**: Dynamic sky colors that transition with each verse

### 📱 Mobile Optimized
- **Responsive Design**: Works beautifully on mobile phones and tablets
- **Touch Navigation**: Swipe gestures for mobile navigation
- **Performance Optimization**: Automatic optimization for lower-end devices
- **Adaptive Quality**: Reduces complexity on mobile for smooth performance

### 🎬 Verse-by-Verse Journey

1. **"The Lord is my shepherd"** - Peaceful green pastures with sheep and trees
2. **"He leads me beside quiet waters"** - Serene water surface with gentle ripples
3. **"He guides me along the right paths"** - Winding path through the landscape
4. **"Through the darkest valley"** - Dramatic mountains with atmospheric fog
5. **"You prepare a table before me"** - Feast table with golden cup and abundance
6. **"I will dwell in the house of the Lord"** - Heavenly temple with pillars and divine light

### 🎵 Interactive Controls
- **Navigation Buttons**: Previous/Next verse controls
- **Auto Play**: Automatically cycles through verses
- **Keyboard Support**: Arrow keys, spacebar, Home/End navigation
- **Touch/Swipe**: Mobile-friendly gesture controls
- **Progress Bar**: Visual indicator of current position

### ⚡ Advanced Features
- **Smooth Animations**: TWEEN-based transitions between verses
- **Performance Monitoring**: Automatic optimization based on device performance
- **Shadow Mapping**: Realistic shadows (disabled on mobile for performance)
- **Tone Mapping**: Enhanced lighting for cinematic quality

## Usage

### For Church Services
This experience is designed as a sermon companion tool. It can be:
- **Projected during sermons** about Psalm 23
- **Used for meditation and reflection**
- **Shared with congregation members** for personal devotion
- **Integrated into worship services** as a visual aid

### Navigation
- **Desktop**: Use arrow keys, spacebar, or click navigation buttons
- **Mobile**: Swipe left/right to navigate, tap buttons for controls
- **Auto Play**: Enable for automatic progression through verses

### Best Practices
1. **Allow loading time** - The 3D assets need a moment to initialize
2. **Use in dimmed lighting** for best visual impact
3. **Consider audio** - Play soft background music for enhanced experience
4. **Test on target devices** - Performance varies by device capability

### Performance Notes

## Customization

### Timing
- **Auto Play interval**: 8 seconds per verse (adjustable in code)
- **Transition duration**: 1.5 seconds between verses
- **Animation speed**: Configurable for different pacing

### Visual Elements
- **Colors**: Sky gradients and lighting can be customized
- **Complexity**: Object counts can be adjusted for performance
- **Effects**: Particle systems and animations can be modified

## Installation

1. **Download all files** to your web server
2. **Ensure Three.js CDN** is accessible
3. **Open index.html** in a web browser
4. **No additional setup required**

## Troubleshooting

### Performance Issues
- **Reduce quality**: The system automatically optimizes for slower devices
- **Check browser support**: Use modern browsers for best experience
- **Clear browser cache**: If experiencing loading issues

### Mobile Issues
- **Rotation**: Works best in landscape mode on phones
- **Touch sensitivity**: Adjust swipe distance if needed
- **Performance**: Older devices may need manual optimization
