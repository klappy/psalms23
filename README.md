# Psalm 23 - An Immersive Journey

An immersive web experience that brings Psalm 23 to life through 3D visuals and spatial audio.

## Features
- 6 unique 3D scenes representing each verse
- Spatial audio with procedural sound generation
- Mobile-optimized responsive design
- Touch gestures and keyboard controls
- Auto-advance through verses
- Customizable settings

## Usage
1. Open index.html in a modern web browser
2. Tap anywhere to begin (mobile) or starts automatically (desktop)
3. Use navigation controls to explore each verse
4. Adjust settings via the gear icon

## Controls
- Touch: Tap/swipe to navigate
- Keyboard: Arrow keys, space bar, M (mute), A (auto-advance), F (fullscreen)
- Mouse: Click to navigate

## Requirements
- Modern web browser with WebGL support
- Web Audio API support
- Recommended: Chrome, Firefox, Safari, or Edge

Perfect for church services, personal devotion, and Bible study.
