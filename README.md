# Psalm 23 - Immersive 2.5D Scrolling Screensaver

A beautiful, immersive 2.5D scrolling background screensaver inspired by Roku's design, specifically crafted for Psalm 23. This experience is optimized for mobile Safari and provides a serene, contemplative environment for reading biblical passages.

## 🌟 Features

### Visual Experience
- **2.5D Parallax Scrolling**: Multiple layers create depth with mountains, hills, valleys, and foreground elements
- **Dynamic Sky**: Animated clouds, glowing sun, and atmospheric lighting effects
- **Pastoral Landscape**: Green pastures, flowing streams, swaying trees, and gentle grass
- **Particle Effects**: Floating atmospheric particles for enhanced immersion
- **Smooth Animations**: 60fps scrolling with optimized performance

### Interactive Controls
- **Pause/Resume**: ⏸/▶ button or spacebar
- **Speed Control**: ⚡ button cycles through 4 speeds (0.5x, 1x, 1.5x, 2x) or use arrow up
- **Fullscreen Mode**: ⛶ button or press 'F' key
- **Touch Optimized**: Responsive controls for mobile devices

### Mobile Safari Optimizations
- **Touch Gestures**: Prevents unwanted scrolling and zooming
- **Performance**: Hardware-accelerated animations and optimized rendering
- **Orientation**: Handles landscape and portrait modes seamlessly
- **Battery Efficient**: Pauses when tab is not visible
- **Accessibility**: Respects reduced motion preferences

## 📱 Mobile Safari Features

- **PWA Ready**: Can be added to home screen
- **Fullscreen API**: Immersive viewing experience
- **Touch Events**: Optimized touch handling
- **Viewport Meta**: Proper scaling and zoom prevention
- **Web App Manifest**: Native app-like experience

## 🎨 Design Philosophy

The visual design reflects the pastoral themes of Psalm 23:
- **Green Pastures**: Lush, rolling hills in various shades of green
- **Quiet Waters**: Animated stream with flowing water effects
- **Mountainous Background**: Distant mountains representing strength and permanence
- **Peaceful Atmosphere**: Gentle animations and calming color palette
- **Spiritual Serenity**: Soft lighting and ethereal particle effects

## 🚀 Getting Started

1. **Open the Experience**: Simply open `index.html` in any modern browser
2. **Mobile Safari**: For the best experience, open in Safari on iOS
3. **Add to Home Screen**: Use Safari's "Add to Home Screen" feature for app-like experience
4. **Fullscreen Mode**: Tap the fullscreen button for immersive viewing

### Performance Optimizations
